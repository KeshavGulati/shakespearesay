# shakepearesay

## Basic Info
This project is built on top of my nanoGPT project that can be found
![here](https://github.com/KeshavGulati/nanoGPT). The inspiration comes from the
popular linux package called *cowsay.*

## How To Run
On bash, run the following commands:

```bash
    git clone https://github.com/KeshavGulati/shakespearesay.com
    cd shakespearesay
    python3 -m venv venv
    source ./venv/bin/activate
    pip install --upgrade pip
    pip install -r requirements.txt
    python3 shakespearesay.py
```
